# ai-ats-analyzer
AI ATS Vision - Smart Resume Analyzer

A full-stack Applicant Tracking System (ATS) analyzer built with Python (Flask) and Gemini 2.5 Flash. This tool allows users to evaluate any resume against a job description, providing a match score, gap analysis, and actionable revision strategies.
